# portfolio-2026
Single Page Application (SPA) portfolio website created for the Portfolio 1 assignment
Includes project showcase, downloadable CV and Cover Letter, optimized thumbnails, and links to all required GitHub repositories and live demos.

## Features
- Single Page Application (SPA) structure
- Responsive mobile-first layout
- Clean typography and accessible structure
- Optimized images for fast loading
- Simple, intuitive navigation

## Technologies used
- HTML
- CSS
- GitHub Pages for hosting

The assignment allows the use of HTML, CSS and optionally TypeScript or JavaScript. For this portfolio project, I chose **not** to include JavaScript because the layout and functionality did not require it. The site is a simple, static Single Page Application without interactive components such as a hamburger menu, dynamic content or scripting‑based navigation. Using only HTML and CSS keeps the code clean, lightweight and fully aligned with the assignment requirements.
